# Security Policy

## Supported versions

Only the latest version published in the catalog manifest is supported.

## Reporting a vulnerability

Please report security issues privately via GitHub's
[Security Advisory form](https://github.com/bmuraykhi/Sub-Jellyfin/security/advisories/new).

**Do not open a public issue for security problems.**

You can expect an initial response within a few days. Once a fix is ready,
it will be released through the normal catalog flow and credited in the
[changelog](CHANGELOG.md) (unless you ask to remain anonymous).
